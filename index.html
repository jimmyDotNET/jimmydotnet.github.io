Vending Machine

This application was part of the Capstone for Tech Elevator's .NET Module 1. My partner, John Bara, and I were tasked with modeling a working vending machine. The application allows users to insert money, view, purchase and consume items from the vending machines inventory. Change is then dispensed in the least amount of coins possible. This application was created using C#.

National Parks

This application was part of the Capstone for Tech Elevator's .NET Module 2. My partner, Michael Weiss, and I were tasked with creating a fully-functional campsite reservation application. It allows you to view parks, campgrounds within parks and avaialable reservations. You can also create reservations. This application was created using C# and SQL Server Express.

Co-Expense Splitter

As a Side Project, I have actively been developing an application for splitting expenses between parties whom may be splitting a large expense (I.E. Rent or other Bills). It also contains features to split restaurant bills and calculate tips, as well as being able to split the tip evenly between several parties.

National Park Geek

This application is another capstone assigned by tech elevator. We are using ASP.NET MVC to create a dynamic, streamlined web app. The end result will be an application for users to look up detailed information on a National Park. It will also display a 5-day Weather Outlook, and suggest how you should prepare per each condition. The users will also be able to take a survey, which will indicate their experience at the selected Park. The parks will then be ranked from highest ratings to lowest.
