Vending Machine

This application was part of the Capstone for Tech Elevator's .NET Module 1. My partner, John Bara, and I were tasked with modeling a working vending machine. The application allows users to insert money, view, purchase and consume items from the vending machines inventory. Change is then dispensed in the least amount of coins possible. This application was created using C#.

National Parks

This application was part of the Capstone for Tech Elevator's .NET Module 2. My partner, Michael Weiss, and I were tasked with creating a fully-functional campsite reservation application. It allows you to view parks, campgrounds within parks and avaialable reservations. You can also create reservations. This application was created using C# and SQL Server Express.
